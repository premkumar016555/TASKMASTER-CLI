# TaskMaster CLI 📝

A simple open-source **To-Do List Command-Line App** in Python.

## 🚀 Features
- Add tasks
- List tasks
- Mark tasks as completed
- Delete tasks
- Stores tasks in `tasks.json`

## 📦 Installation
```bash
git clone https://github.com/your-username/taskmaster.git
cd taskmaster
python taskmaster.py
